# DCF Valuation Skill

Institutional-grade DCF (Discounted Cash Flow) valuation models with comprehensive share dilution analysis.

## Files

- `SKILL.md` - Main skill file (upload to Claude Skills)
- `Share_Dilution_Analysis_Knowledge.md` - Supporting knowledge file (upload to Project Knowledge)

## Installation

1. **Upload SKILL.md** to your Claude Skills folder
2. **Upload Share_Dilution_Analysis_Knowledge.md** to your Project Knowledge

The skill references the Knowledge file for detailed dilution methodology, formulas, and templates.

## Features

- Treasury Stock Method (TSM) for options/warrants
- If-Converted method for convertible securities
- RSU/PSU dilution treatment
- SBC-adjusted Free Cash Flow
- Multi-tab Excel workbook output
- Sensitivity and scenario analysis
- Sector-specific guidance (Tech, Biotech)

## Usage

Ask Claude to:
- "Build a DCF model for [TICKER]"
- "Value [COMPANY] with full dilution analysis"
- "Calculate fully diluted shares for [TICKER]"
